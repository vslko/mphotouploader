mPhotoUploader demo
Features
  • Select image by one click
  • Image changing at once after upload
  • Display image by default
  • Change image without server request
  • Additional parameters in request to server image uploading
Getting started
Insert in head
<!-- basic style -->
<link rel="stylesheet" type="text/css" href="your_path/css/mphotouploader.css" />
<!-- required javascripts -->
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>
<script type="text/javascript" src="your_path/js/mphotouploader.js"></script>
Insert in body
<div id="my-photo"></div>

<script type="text/javascript">
var photo = $('#my-photo').mPhotoUploader({
	url	: './server/upload.php',
	name	: 'pic',
	photo	: './img/1.jpg',
	onError	: function( error ) { alert(' Error: ' + error); }
});
</script>
Documentation
Options
urlType: string, Default: "./upload/" Path to server script for upload image. nameType: string, Default: "myfile" Name of image-variable in request. photoType: string, Default: null Path to image on initialization. paramsType: object, Default: {} Additional parameters in request. localeType: object, Default: in english Translations. Contents such fields:
exten_error : 'File {filename} is not image',
fatal_error : 'Fatal error'
Methods
object.mPhotoUploader('setPhoto', string/null )Arguments: path to image Set new image without server request, if path is null, then show default image object.mPhotoUploader('setParams', object )Arguments: object of parameters Set additional parameters
Events
onSuccess( path ) Triggered when image is uploaded. Contents path to uploaded image on server onError() Triggered when image uploading is failed. Contents error message
